## 출력이 많은 경우

c++의 경우 줄바꿈을 위하여 endl을 사용하는 것보다 '\n'을 사용하는 것이 더욱 빠르다.

```c++
// 변수 answer를 10만번 출력하는 경우
for(int i = 0; i < n; i++){
    // 정답을 구하기 위한 코드
    ...
    
    // endl 보다 '\n'이 실행시간이 빠르다
    // cout << answer << endl;
    cout << answer << '\n';
}
```
